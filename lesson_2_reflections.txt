What happens when you initialise a repository? Why do you need
to do it?

Git takes a snapshot of the current files. You need to do it so
you can start using Git

How is the staging area different from the working directory and the
repository? What value do you think it offers?

It is a middle between the two so you can make sure you want to commit before you do.

How can you use the staging area to make sure you have one commit per logical change?

You can add as many files as you need to the staging area to make sure that you have only one commit per logical change
